Installation:

Inside the archieve you will find the following:

  • HKLocalTools.mel file.
  • MenuItem folder containing needed files to add LT to your Modify/Transform tools menu for versions 2008 and below.
  • The icon

To install the script:

  • Place the HKLocalTools.mel file to the local directory Documents and Settings\"Username"\My Documents\maya\"version"\scripts.
  • Open up you userSetup.mel located at MyDocuments\maya\"version"\scripts and add this line "source HKLocalTools; ".
    If you don't have this userSetup file - create it..
  • That's it, open up maya, you are now ready to use the tool! :-).

For assigning the tool to hotkey/shelf, positioning the hud please refer to the customization page.
